# AI Travel Planner with Budget Optimization

## Description
AI Travel Planner is a web application that helps users plan their trips easily based on destination and budget. It uses AI to generate itinerary, hotel suggestions, food recommendations, clothing suggestions, travel tips, and estimated cost breakdown.

## Features
- Generate AI-based travel itinerary
- Budget-based hotel suggestions
- Food recommendations
- Estimated cost breakdown
- Clothing suggestions based on destination and weather
- Useful travel tips
- Simple and user-friendly interface

## Technologies Used
- Python
- Flask
- HTML, CSS
- Gemini AI API
- Git and GitHub

## How to Run

1. Install Python (Make sure Python 3 is installed)

2. Clone the repository:
git clone https://github.com/ManishaVallapu/ai-travel-planner.git

3. Navigate into the project folder:
cd ai-travel-planner

4. Install required libraries:
pip install flask python-dotenv google-generativeai markdown

5. Create a .env file in the project folder and add:
GEMINI_API_KEY=your_api_key_here

6. Run the application:
python app.py

7. Open your browser and go to:
http://127.0.0.1:5000
